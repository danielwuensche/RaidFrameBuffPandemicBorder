# RaidFrameBuffPandemicBorder

This World of Warcraft addon adds a border to buff icons of party frames, if the remaining buff duration is inside the pandemic window.
It isn't fancy, but gets the job done.

I'm playing Resto Druid and don't like switching out the whole party/raid frames, so I made this addon since this was what I was missing most.

### What's the pandemic window?

If a buff is refreshed when it has at most 30% of its original duration left, the remaining duration is added onto the new buff duration. When refreshing outside of the pandemic window, the remaining duration gets lost.
